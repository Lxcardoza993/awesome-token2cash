# 📈 Awesome Token2Cash

**Languages:** [English](README.md) • [简体中文](README.zh-CN.md) • [Español](README.es.md) • [日本語](README.ja.md) • **한국어** • [Français](README.fr.md) • [Русский](README.ru.md)

> 주식 분석, 트레이딩, 금융 학습을 위한 **멀티 에이전트 및 대규모 언어 모델(LLM) 플랫폼**의 선별 리스트입니다.
> 거인의 어깨 위에 서서, 이 분야를 체계적으로 정리합니다 — 바퀴를 재발명하지 않고도 학습하고, 비교하고, 직접 만들 수 있도록 돕습니다.
>
> 각 항목에는 ⭐ 스타 수, 주요 언어, 라이선스가 표시되어 어떤 프로젝트가 학습 가치가 있는지 빠르게 판단할 수 있습니다.

- **멀티 에이전트 토론 / 역할 전문화** — 분석을 전문적인 페르소나(매수, 매도, 리스크 관리자, 포트폴리오 매니저)로 나누고, 구조화된 토론을 통해 이견을 해결합니다. (예시: TradingAgents, orallexa-ai-trading-agent, ai-hedge-fund)
- **공시 대상 검색 증강 / 금융 RAG** — 검색 증강을 통해 SEC 공시, 실적 발표 원고, 시장 데이터에 LLM 출력을 근거시킵니다. (예시: FinRAG, openbb-rag-financial-research-agent, multi-horizon-financial-agent)
- **LLM 기반 감성-신호 파이프라인** — 뉴스, 소셜 미디어, 실적 발표 감성을 정량적 트레이딩 신호로 전환합니다. (예시: twitter-alpha-sentiment-tracker-v2, FinNLP, ProsusAI/finBERT)
- **금융 코퍼스용 LoRA/PEFT 파인튜닝 레시피** — 실적 발표, 뉴스, 보고서 등을 활용한 경량 파인튜닝으로 범용 LLM을 금융 분야에 맞춥니다. (예시: FinGPT, FinLoRA, PIXIU)
- **메모리 증강 트레이딩 에이전트** — 단기 및 장기 기억을 에이전트에 추가하여 포트폴리오 맥락을 유지하고 행동을 적응시킵니다. (예시: FinMem-LLM-StockTrading)
- **리서치 보고서 생성을 위한 ReAct + 도구 사용** — 추론과 행동 루프를 결합한 LLM으로 데이터를 수집하고 투자 리서치 보고서를 작성합니다. (예시: deep-research-machine, stock_research_agent)
- **백테스팅 + 페이퍼 트레이딩 샌드박스** — 실제 배포 전에 과거 시장 데이터 재현과 브로커 페이퍼 계정으로 에이전트 전략을 검증합니다. (예시: FinRL, Qbot, AlpacaTradingAgent)
- **리스크 가드 / 인간이 포함된 거부권** — 신호 생성과 실제 거래 실행 사이에 가드레일 또는 인간 승인 게이트를 둡니다. (예시: AutoTrader-AgentEdge, ai-hedge-fund, LLM-TradeBot)
- **멀티모달 시장 이해** — 주가 차트, 대체 데이터, 텍스트를 결합하여 주식 분석을 개선합니다. (예시: StockerLens, FinAgent)
- **생성형 에이전트를 활용한 시장 시뮬레이션** — LLM 에이전트로 경제/트레이딩 주체 집단을 시뮬레이션하여 시장 역동성의 출현을 연구합니다. (예시: TwinMarket, LLM-Economist, ai-economist)
- **금융 벤치마크 중심 개발** — 금융 특화 QA, 임베딩, 트레이딩 벤치마크를 활용해 LLM/에이전트 역량을 평가합니다. (예시: FinEval, financebench, BizFinBench)
